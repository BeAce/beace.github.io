最好用的markdown编辑器汇总

November 15, 2016

收集了一下好用的markdown编辑器，都曾经大量使用，都爱不释手。选择恐惧症。。。

1. stackedit

  • 在线编辑器
  • 支持同步google drive，github等

beta版

2. dillinger

  • 在线编辑器

  • 支持同步google drive，Dropbox

  • 更有意思的是，你可以下载它的源码到本地，启动它运行。利于搭建静态博客。
$ cd dillinger
$ npm install -d
$ node app

3. 作业部落

  • 在线编辑器+客户端
  • 部分功能付费

作业部落客户端

4. MEditor.md

  • 开源在线markdown编辑器

5. markdownpad

  • 客户端

  • 只支持Windows

6. MOU

mou是非常轻量级的markdown编辑器，为止到现在唯一让我体验不好的地方就是不支持目录的功能。

7. 还有一些字体看起来不太友好的

那么问题来了，你在用哪一款呢？
