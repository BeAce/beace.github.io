如何科学上网-想要翻墙怎么办

March 22, 2017

首先通过hosts文件翻墙。

具体方法见 https://github.com/racaljk/hosts。

这里简述下Mac下的操作。

将 https://raw.githubusercontent.com/racaljk/hosts/master/hosts 链接中的hosts的文件内容复制，mac下进行如下命令：

删除本地hosts文件（如果重要需要备份），在终端最初始目录进行操作。

$ sudo rm -f /etc/hosts

新建hosts文件，将刚才复制内容粘贴

$ sudo vi /etc/hosts

hosts文件如下图所示

更新完成后即可进入谷歌搜索，请使用https的方式。例如：https://google.com.hk 访问。

更新完成后大部分常见网站都可访问，例如Facebook、Twitter、youtube等，少数被墙的厉害的难以进入，但是想要在youtube上看个视频还是有点卡的。

虽然完成了翻墙这个操作，但是访问速度还是比较慢，但是对于不是程序员的大哥大姐已经很够用了。下面说一种快的方式。

XX-net

XX-net在使用时需要梯子或者通过上述方式翻墙后进行。 关于XX-net的使用方式，其文档已经很详细很详细了，而且中文版的更详细，每一个步骤都有截图。

在这里，说几个比较不容易理解的问题。

申请Google appid，根据教程里面的链接步骤申请appid，申请appid需要进入Google的控制台（console），这也是为什么会先替换hosts。

appid本来作为Google开发者使用其API或SDK进行开发使用，XX-net通过使用GoAgent用来代理本地的网络请求。

在使用XX-net经常会碰到“请检查浏览器代理设置”的问题。

建议安装SwitchyOmega插件来进行自动切换代理。避免开启代理后国内网站访问速度慢的问题。选择XX-Net自动切换。
